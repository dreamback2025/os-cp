# TaskScheduler - 进程/线程调度模拟器

这是一个为Windows平台设计的多道程序设计技术项目，实现了时间片轮转调度算法。

## 项目概述

本项目实现了操作系统中的多任务并发运行机制，具体包括：
- 时间片轮转（Round Robin）调度算法
- CPU现场保护与恢复
- 手工输入和自动产生任务两种方式
- 键盘控制时间流逝
- 执行日志记录与重放功能

## 项目文件

- `scheduler_simulator.c`: 主程序源代码
- `TaskScheduler.sln`: Visual Studio解决方案文件
- `TaskScheduler.vcxproj`: Visual Studio项目文件
- `README.md`: 本说明文件

## 如何在Visual Studio中使用

1. 打开Visual Studio
2. 选择 "文件" -> "打开" -> "项目/解决方案"
3. 导航到项目目录并选择 `TaskScheduler.sln`
4. 解决方案将自动加载项目
5. 按F5编译并运行程序

## 程序功能

运行程序后，您将看到以下选项：

1. **手工输入任务** - 手动创建任务，指定名称、优先级、执行时间和到达时间
2. **自动产生任务** - 系统自动生成测试任务
3. **运行调度器** - 按任意键前进一个时间单位，观察任务调度过程
4. **显示任务列表** - 查看当前所有任务信息
5. **显示执行日志** - 查看调度过程中的详细日志
6. **保存执行日志到文件** - 将调度结果保存到execution_log.txt
7. **从文件加载执行日志** - 从文件中加载之前保存的调度日志
8. **重放执行日志** - 重放之前保存的调度过程
9. **退出** - 退出程序

## 系统要求

- Windows操作系统
- Visual Studio 2022或更高版本
- 支持C语言的编译环境
